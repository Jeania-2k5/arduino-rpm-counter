Arduino RPM Counter Project

Measure Rotations Per Minute with Arduino + IR Sensor + LCD

Project Overview

This project is a digital tachometer using an Arduino Uno, IR sensor, and I2C LCD. It detects object rotations and displays the RPM in real-time.

Components Used

  • Arduino Uno
  • IR Sensor
  • I2C 16x2 LCD
  • Jumper Wires

How It Works

The IR sensor detects each rotation via a reflective tape. Arduino counts these pulses for 1 minute and calculates RPM. Results are shown on the LCD and serial monitor.

Output Example

LCD Display: Final RPM: 120

Serial Monitor: Rotations: 2, 5, 9... Final RPM: 120

Applications

This RPM counter can be used in automotive engines, robotics, industrial motors, and educational labs.
